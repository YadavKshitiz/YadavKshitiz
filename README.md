## 🚀 Projects

### [WeatherSnap](https://github.com/YadavKshitiz/WeatherSnap) — Android Weather Report App
> Kotlin · Jetpack Compose · MVVM · Hilt · Retrofit · Room DB · CameraX · Coroutines · StateFlow

A full Android app built solo. Fetches live weather via the Open-Meteo REST API, captures and compresses images through a custom CameraX interface built from scratch, and persists structured reports locally using Room DB. Solved a SavedStateHandle lifecycle bug that caused data loss on device rotation.

---

### [FundusCare](https://github.com/YadavKshitiz/FundusCare) — Medical Imaging Android App
> Kotlin · Jetpack Compose · REST API · Android SDK

Android app built for an academic project. Users capture or select a retinal image, upload it to an ML backend via REST API, and receive an AI-generated report on the detected stage of a fundus eye disease. Includes a scan history screen tracking every upload with its timestamp and diagnosis.

---

### [Calmi](https://github.com/YadavKshitiz/Stress_Detection_App) — Stress Detection App
> Kotlin · Jetpack Compose · REST API · AI Chatbot Integration

Built the entire Android layer solo in a 4-person team. The app captures live camera and audio, sends it to a custom ML backend that detects stress levels with 70–80% accuracy, and responds through an AI chatbot with personalised suggestions.
